◆ generate_random_mac()

void generate_random_mac ( uint8_t * mac)
static

Gera um endereço MAC pseudo-aleatório através do /dev/urandom.

  • Aplica as máscaras de bits necessárias para garantir que o MAC gerado está em conformidade com as regras de endereço Unicast Localmente Administrado (ULA).
  • Parâmetros
    macBuffer de 6 bytes de saída para o endereço MAC.
