Valida a tag HMAC e, se fidedigna, desencripta o conteúdo cifrado.

Parâmetros
ciphertextPonteiro para a sequência de dados em formato cifrado.
lenComprimento do bloco cifrado em bytes.
ivPonteiro para o Vetor de Inicialização usado na cifragem.
hmacPonteiro para a assinatura recebida a ser auditada.
plaintextBuffer de saída onde o texto original decifrado será escrito.
Retorna
1 se o HMAC for autêntico e a decifragem concluir, 0 se o pacote for corrompido/adulterado.
